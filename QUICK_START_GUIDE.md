## Common Tasks

### How to Add a Subject?
```
1. Admin Panel ‚Üí Click "Subjects" tab
2. Click "Add Subject"
3. Select: Semester, then enter name
4. Click "Add" button
```

### How to Reorganize Faculty?
```
1. Admin Panel ‚Üí Click "Faculties" tab
2. Find the faculty
3. Click "Edit" button
4. Modify the name
5. Click "Save" button
```

### How to Archive a Post?
```
1. Browse ‚Üí Click post
2. Click "Edit"
3. Uncheck "Publish immediately"
4. Click "Save Changes"
‚Üí Post becomes draft (archived)
```

### How to Promote a Draft?
```
1. Browse ‚Üí Filter by "Draft" status
2. Click post
3. Click "Edit"
4. Check "Publish immediately"
5. Click "Save Changes"
‚Üí Post becomes published
```
